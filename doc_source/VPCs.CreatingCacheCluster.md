# Creating a Cache Cluster in an Amazon VPC<a name="VPCs.CreatingCacheCluster"></a>

In this example, you create a cache cluster in your Amazon VPC\.

## Creating a Cache Cluster in an Amazon VPC \(Console\)<a name="VPCs.CreatingCacheCluster.CON"></a>

To launch a Redis cache cluster, see [Creating a Redis \(cluster mode disabled\) Cluster \(Console\)](Clusters.Create.CON.Redis.md)\. In step 6\.d select a VPC subnet group\.

You have now launched a cache cluster inside an Amazon VPC\. For an example of one way to connect to your new cache cluster running in the Amazon VPC, continue to [Connecting to a Cluster or Replication Group Running in an Amazon VPC](VPCs.Connecting.md)\.
